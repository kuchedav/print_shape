# print_structure

## Description
Print structure of any arbitrary python object.

## Installation

Run the following to install:
```
pip install print_structure
```

## Attributes
object : any type of python object

## print out
Structure of object.

## Usage
```
from print_structure import print_str

list = [1,2,3,["a","b","c"]]
print_str(list)
```
output
```
|___0 [list] (4)
  |___1 [list] (3)
```
